## Features
- ⚡️ Vue 3, Vite 2, pnpm, ESBuild, Vitepress
- 💪 Vue 3 Composition API
- 🔥 Written in TypeScript
- 🗂 File based routing
- ⚙️ Unit Testing with Vitest
- 😃 Eslint + Prittier
- 🎨 UnoCSS - the instant on-demand atomic CSS engine
- 🌍 I18n ready
- 🚘 CI/CD with GithubActions
